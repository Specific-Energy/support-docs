Built with: Mkdocs

Getting Started

To get a local copy up and running follow these simple steps.

Generating a GitHub SSH key

More info from GitHub

  1. Create a passphrase using the random password generator of your choice.

  2. Generate an ssh key
    ssh-keygen -t ed25519 -C "your_email@example.com"

  3. Start the ssh-agent in the background.
    eval "$(ssh-agent -s)"

  4. Add your SSH key to the ssh-agent
    ssh-add ~/.ssh/id_ed25519

  5. Add the SSH key to your account on GitHub

Installation

  1. Clone the repo
    git clone git@github.com:Specific-Energy/support-docs.git

  2. Run the init script ./init.sh

Usage

  • The main folder to edit is docs/
    • Each page is an .md file

To add a new page:

  1. Add the page as a .md file in docs
  2. Add the page in the mkdocs.yml under nav:

To add a nested page:

  1. Add the page as a .md file in docs under a folder
    • Ex: components/containers.md
  2. Add the page in the mkdocs.yml under nav:


To run the project locally:
mkdocs serve

To build the site:
mkdocs build

To deploy:
mkdocs gh-deploy
